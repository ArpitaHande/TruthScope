# TruthScope
AI Transparency and Bias Analysis platform

TruthScope is a web-based AI auditing tool that enhances responses from AI systems like Google Gemini by analyzing them for bias, oversimplification, and lack of transparency.

When a user enters a query, the system first generates an AI response and then applies a custom analysis layer that evaluates the reasoning behind that answer. It highlights potential issues such as reliance on limited metrics (like IQ), repeated or homogeneous sources, and missing perspectives. The tool then presents the user with both the original response and clear “Truth Insights” that warn about possible biases and suggest more balanced interpretations.

How it Solves the Problem

Many AI systems provide confident answers without explaining their limitations, which can lead users to trust incomplete or biased information. TruthScope addresses this by:

Improving transparency → showing how the answer might be biased
Reducing blind trust → alerting users to limitations in reasoning
Encouraging critical thinking → presenting alternative perspectives
Promoting fairness → highlighting underrepresented viewpoints

By acting as an AI audit layer, TruthScope transforms AI from a “black box” into a more interpretable and responsible system, helping users make better-informed decisions.
